# Calculadora de IMC

Este é um programa simples em Java que calcula o Índice de Massa Corporal (IMC) de uma pessoa.

## Funcionalidades

- Cálculo do IMC baseado em peso e altura
- Classificação do IMC em diferentes categorias:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade grau 1
  - Obesidade grau 2
  - Obesidade grau 3
